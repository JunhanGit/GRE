快速上手：把 GRE 词汇训练器用在 iPhone 上

你需要准备： 一个 GitHub 账号（或 Netlify 账号），以及解压后的项目文件夹。

方法 A：GitHub Pages（推荐）

  1. 在 GitHub 新建一个仓库（任意名字，例如 gre-vocab）。
  2. 把本项目解压后的所有文件拷贝到仓库根目录并提交。
  3. 进入仓库的 Settings → Pages，将 Source 设为 Deploy from a branch，分支选 main，目录选 /root，保存。
  4. 等 1 分钟，页面会显示你的访问地址（形如 https://用户名.github.io/仓库名/）。
  5. 用 iPhone 的 Safari 打开该地址 → 点击底部 分享 → 添加到主屏幕。

方法 B：Netlify（更省事）

  1. 登录 Netlify，点击 Add new site → Deploy manually。
  2. 把解压后的项目文件夹拖进去，部署结束后会得到一个 .netlify.app 域名。
  3. 用 iPhone 的 Safari 打开该域名 → 分享 → 添加到主屏幕。

常见问题

  • 离线能用吗？ 首次联网打开后会缓存，之后可离线使用。
  • 发音没声？ iOS 需要先手点一次发音按钮授权音频，之后正常。
  • 如何更新到新版？ 本包的 Service Worker 缓存名是 gre-pwa-v4-21a。新版会换名，刷新两次即可更新。
